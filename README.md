# Medical Cost Prediction System

A Machine Learning based web application that predicts estimated medical insurance costs using patient demographics, lifestyle, and healthcare-related factors. This project helps insurance companies improve premium pricing, risk assessment, and decision-making through data-driven predictions.

## 🚀 Features

* Predict medical insurance cost instantly
* FastAPI backend for high performance
* Clean and responsive UI using HTML & CSS
* Machine Learning regression model
* User-friendly web form
* Ready for deployment on Render / Cloud

## 🛠️ Tech Stack

* Python
* FastAPI
* HTML5
* CSS3
* Pandas
* Scikit-learn
* Jinja2
* Uvicorn

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/your-username/Medical-Cost-Prediction-System.git
cd Medical-Cost-Prediction-System
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Application

```bash
uvicorn app:app --reload
```
